# GitHub Setup Instructions

Follow these steps to push your code to GitHub:

1. Create a new repository on GitHub:
   - Go to https://github.com
   - Click the "New" button to create a new repository
   - Name it "fidenz-weather-app"
   - Make it public or private as you prefer
   - Do NOT initialize with a README, .gitignore, or license
   - Click "Create repository"

2. In your terminal, run these commands:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/fidenz-weather-app.git
   git branch -M main
   git push -u origin main
   ```

3. To add the required collaborators:
   - Go to your repository page on GitHub
   - Click on "Settings" tab
   - Click on "Collaborators & teams" in the left sidebar
   - Click "Add people" and add these GitHub usernames:
     - kanishka.d@fidenz.com
     - srimal.w@fidenz.com
     - narada.a@fidenz.com
     - amindu.l@fidenz.com

Replace YOUR_USERNAME with your actual GitHub username.
